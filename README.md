# Grid format

Grid format is a Moodle course format that displays course sections as responsive cards with cover images, completion indicators, and activity navigation support.

## Requirements

- Moodle 4.5 or later
- PHP version supported by the target Moodle release

## Installation

1. Copy this plugin to `course/format/grid`.
2. Visit site administration to complete the Moodle plugin installation.
3. Set a course format to **Grid format** from the course settings page.

## Repository

The Moodle Plugins Directory recommends the repository naming pattern `moodle-{plugintype}_{pluginname}`. This plugin is published as:
